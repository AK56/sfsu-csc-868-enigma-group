Dont open the sql file in any word processing software.

Steps to import the data.
1. Start workbench
2. Log in to localhost 3306
3. Click on Server -> Data Import
4. Select 'Import from Self_Contained File' and select the SQL file
5. Click on 'New...' and choose name as monopoly
6. At the bottom, select 'Dump Structure and Data'
7. Click on Start Import

This will take some time.
